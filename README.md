📊 Store Data Analysis using Excel
📌 Project Overview

This project focuses on analyzing store sales data using Microsoft Excel to generate meaningful insights and support business decision-making. The analysis includes sales performance, customer behavior, product trends, and regional insights.


🎯 Objectives
1.Analyze overall store performance
2.Identify top-selling products and categories
3.Understand customer purchase patterns
4.Evaluate sales trends over time
5.Provide actionable insights using data visualization

🛠️ Tools & Technologies
1.Microsoft Excel
2.Pivot Tables
3.Pivot Charts
4.Data Cleaning & Formatting
5.Conditional Formatting
6.Excel Functions (SUM, AVERAGE, IF, VLOOKUP, etc.)

📂 Dataset Description

The dataset contains the following fields:
1.Order ID
2.Order Date
3.Customer Name
4.Region
5.Product Category
6.Product Name
7.Quantity Sold
8.Sales Amount
9.Profit

🔍 Key Analysis Performed
📅 Monthly & Yearly Sales Analysis
🛍️ Category-wise Sales Performance
🌍 Region-wise Sales Distribution
👥 Customer Purchase Trends
💰 Profit Analysis

📈 Dashboard Features
1.Interactive filters (Slicers)
2.Dynamic charts and graphs
3.KPI indicators (Total Sales, Profit, Orders)
4.Easy-to-understand visual insights

Conclusion

This project demonstrates how Excel can be used as a powerful tool for data analysis and visualization. It helps in transforming raw data into actionable insights for better business decisions.
